## Summary

Build a personalized nutrition planner using a Neuro-Symbolic architecture. The application uses a Multi-Layer Perceptron (MLP) to calculate daily caloric needs based on a user's physical profile, filters a food database using a Knowledge-Based System (KBS) for allergy safety, and selects an optimized meal plan using Optimization Theory (OT).

## Technical Context

**Language/Version**: Python 3.11
**Primary Dependencies**: Streamlit, TensorFlow/Keras, Pandas, Scipy
**Storage**: CSV (Kaggle Diet Recommendations dataset)
**Testing**: pytest
**Target Platform**: Web (Streamlit)
**Project Type**: Web Application
**Performance Goals**: Inference < 200ms, Safety Filtering < 100ms
**Constraints**: 
- Must follow Neuro-Symbolic flow: NN (Prediction) -> KBS (Safety) -> OT (Selection)
- Must use Streamlit sidebar for inputs and `st.dataframe` for output visualization.
